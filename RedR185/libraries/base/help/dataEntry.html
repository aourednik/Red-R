Data Entry

Brief

A table input data entry into a data.frame.

Details

Similar to Excel, you can enter data into a table. A data.frame is created in R on submit and sent forward.

Parameters

Data Table
input data as needed.
Add column
add additional columns of data input. On click requires input of column name.

Inputs

None

Outputs

base:RDataFrame
