Introduction

Shows the output of an R variable, equivalent to typing the variable name in the R Executor

Details

Widgets may send forward many types of data.  These may not always be conducive to viewing in a table viewer such as RDataTable.  This widget shows the output of any Red-R object.  This can be useful as a quick way to view data and to check the integrity of data.  This widget has a block on showing data from a table larger than 500 rows.  If then row length is larger than 500 only the first 500 rows are shown.
