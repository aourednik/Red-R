Introduction

Separates variables from an environment and sends them. Generally used with the R Loader Widget.

Details

Data loaded from an R file is first loaded using the R Loader.  Variables from that session can be separated using this widget.  All variables are displayed in a list.  Clicking an item in the list will send that data element into the schema through the slot coresponding to it's data type.  Data Frames, Lists, and Vectors are preferred over the R Variable type.
